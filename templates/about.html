About JobGuard AI

A college mini-project that uses Machine Learning and rule-based analysis to detect fraudulent job and internship postings in real-time.

ML-Powered Detection
Uses a trained Logistic Regression model with TF-IDF vectorization to classify job postings based on patterns learned from thousands of examples.
OCR Image Analysis
Upload a screenshot of a job offer. Tesseract OCR extracts the text automatically so you can analyze WhatsApp forwards and image-based offers.
Rule-Based Scoring
Over 30 hand-crafted rules that detect scam patterns: payment requests, unrealistic salaries, WhatsApp recruiting, urgency tactics, and more.
Explainable Output
Not just a prediction — but a full report with risk level, detailed reasons, system insights, and actionable recommendations for the user.

Technology Stack

Python 3 Flask Scikit-Learn Logistic Regression TF-IDF Vectorizer Tesseract OCR pytesseract Pillow HTML5 / CSS3 Vanilla JavaScript Fetch API
